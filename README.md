kommuneplanScraper
======

A script for extracting information of granted exemptions from [Karm√∏y Kommune kommuneplan](http://webhotel2.gisline.no/gislinewebplan_1149/gl_planarkiv.aspx?planid=670), and plotting the coordinates in a Google Map.

Usage
-----
Run the script

    python scrape.py

The script sometimes fails, if so just try again. It will usually work the second time. Feel free to try debugging this.
